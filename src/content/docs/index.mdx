## Quick Start

Get your site running in about 10 minutes:

```bash
# Clone the repository
git clone https://github.com/vincentheimann/astro-swiss-starter-theme.git
cd astro-swiss-starter-theme

# Install dependencies
npm install

# Start development server
npm run dev
```

Then customize your brand colors, company info, and content. **[Follow the complete guide ‚Üí](/getting-started/)**
